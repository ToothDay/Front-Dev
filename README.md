# 내손안의 치아 ToothDay


"내손안에 치아" 프로젝트는 사용자에게 치아 건강을 관리하고, 치과 예약을 보다 간편하게 볼 수 있는 서비스입니다.

## 프로젝트 개요

- **개발 기간**: 2023년 6월 26일 - 7월 28일
- **팀 구성**: 
  - 백엔드 개발자: 3명
  - 프론트엔드 개발자: 2명
  - 디자이너: 1명
  - 기획자: 1명
- **기술 스택**:
  - **프론트엔드**: Next.js, TypeScript, Zustand, React Query, SCSS 모듈, Framer Motion
  - Node 버전 (v20)

## 프로젝트 실행

### 환경설정
`.env` 파일이 필요합니다

### 실행

1. 프로젝트 클론
2. 패키지 설치

```
npm install
```

3. 로컬 서버 실행
```
npm run dev
```
